# PROPUESTA DE PROYECTO FINAL

## TÍTULO DEL PROYECTO
**Tolio: Plataforma Web de Economía Colaborativa para Préstamo de Herramientas y Servicios**

## 1. DESCRIPCIÓN DEL PROYECTO

### 1.1 Contexto y Motivación

En la actualidad, muchas herramientas y equipos son utilizados esporádicamente pero requieren una inversión económica significativa. Simultáneamente, existe un creciente interés en modelos de economía colaborativa que permiten optimizar el uso de recursos y generar ingresos adicionales para quienes poseen estos bienes.

El proyecto Tolio surge como una solución a esta problemática, creando un espacio digital donde las personas pueden ofrecer sus herramientas, equipos y servicios en alquiler, mientras que otros pueden acceder a estos recursos cuando los necesitan, sin necesidad de adquirirlos permanentemente.

### 1.2 Descripción General

Tolio es una plataforma web que funciona como marketplace bidireccional, conectando a propietarios de herramientas y prestadores de servicios con usuarios que necesitan acceder temporalmente a estos recursos. La plataforma facilita todo el proceso desde la publicación de ítems, búsqueda y reserva, hasta el pago y la gestión de transacciones.

**Funcionalidades principales:**

- **Para Propietarios/Prestadores:**
  - Publicación de herramientas, equipos o servicios disponibles para alquiler
  - Establecimiento de precios y condiciones de uso
  - Gestión de reservas y calendarios de disponibilidad
  - Sistema de cobros y seguimiento de ingresos
  - Protección mediante depósitos de seguridad

- **Para Usuarios/Solicitantes:**
  - Búsqueda y navegación de ítems por categorías
  - Sistema de reservas con selección de fechas
  - Múltiples opciones de pago seguro
  - Sistema de reseñas y valoraciones
  - Comunicación directa con propietarios

- **Características de Seguridad:**
  - Verificación de identidad mediante reconocimiento facial
  - Sistema de pagos seguros con múltiples procesadores (Stripe, MercadoPago, Blockchain)
  - Depósitos de seguridad para protección de propietarios
  - Sistema de resolución de disputas
  - Historial de transacciones y reputación de usuarios

### 1.3 Alcance

El proyecto incluye el desarrollo completo de una aplicación web responsive que permite:

1. Registro y autenticación de usuarios con verificación de identidad
2. Gestión completa del ciclo de vida de publicaciones (crear, editar, eliminar)
3. Motor de búsqueda y filtrado de ítems disponibles
4. Sistema de reservas con gestión de disponibilidad
5. Integración con múltiples pasarelas de pago
6. Sistema de mensajería entre usuarios
7. Panel de administración para propietarios y usuarios
8. Sistema de reseñas y reputación
9. Notificaciones en tiempo real
10. Dashboard con estadísticas y métricas

### 1.4 Objetivos

**Objetivos Generales:**
- Desarrollar una plataforma web funcional y escalable para economía colaborativa
- Facilitar el acceso a herramientas y servicios mediante un modelo de alquiler
- Crear un ecosistema seguro y confiable para transacciones entre usuarios

**Objetivos Específicos:**
- Implementar un sistema de autenticación robusto con verificación biométrica
- Integrar múltiples procesadores de pago para mayor flexibilidad
- Diseñar una interfaz intuitiva y responsiva para diversos dispositivos
- Garantizar la seguridad de las transacciones mediante depósitos y escrow
- Desarrollar un sistema de reputación basado en reseñas verificadas
- Optimizar la experiencia de usuario mediante notificaciones y mensajería

### 1.5 Justificación Académica

Este proyecto integra múltiples áreas de las ciencias de la computación:

- **Ingeniería de Software:** Arquitectura de aplicaciones web, patrones de diseño, gestión de estado
- **Bases de Datos:** Modelado de datos relacionales, optimización de consultas
- **Seguridad Informática:** Autenticación, autorización, verificación biométrica, procesamiento seguro de pagos
- **Interfaces de Usuario:** Diseño UX/UI, accesibilidad, responsive design
- **Desarrollo Web Full-Stack:** Frontend, backend, APIs RESTful
- **Integración de Sistemas:** APIs de terceros, procesadores de pago, servicios externos

El proyecto representa un desafío técnico significativo que requiere la aplicación práctica de conocimientos adquiridos durante la carrera, al mismo tiempo que aborda una problemática real con impacto social y económico.
